# Kalkulator Resep Minuman 🧋

Program C++ untuk menghitung dan mengkonversi resep minuman berdasarkan jumlah yang diinginkan.

## Fitur

- Menu utama menampilkan 6 pilihan resep
- Input jumlah resep (bisa desimal: 0.5, 1, 2, 3, dst.)
- Menampilkan detail bahan yang dibutuhkan
- Konversi otomatis dari pack ke gram
- Estimasi volume total dan jumlah gelas

## Resep yang Tersedia

| No | Nama Resep | Volume Dasar |
|----|-----------|-------------|
| 1 | Base Teh | 3000 ml |
| 2 | Biang Orange | 5000 ml |
| 3 | Biang Lemon Tea | 4000 ml |
| 4 | Base Thai Tea | 2000 ml |
| 5 | Biang Milo | 2240 ml |
| 6 | Biang Green Tea | 3260 ml |
